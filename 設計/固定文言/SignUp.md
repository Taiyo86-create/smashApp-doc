# SignUp画面の固定文言

| 項目               | 説明                                 |
|--------------------|--------------------------------------|
| **名前**           | 登録する名前を入力してください。       |
| **プロフィール**   | アクティブな時間やひと言を記述しましょう        |
| **Email**          | 有効なメールアドレスを入力してください。 |
| **パスワード**     | パスワードを設定してください。(6文字以上の英数字記号)   |
| **使用キャラ**     | 1番使えるキャラクターを選択してください。 |
| **レート帯**       | あなたのレート帯を選択してください。   |
| **戦いたいキャラ** | 戦いたいキャラクターを選択してください。 |
| **登録ボタン**     | 登録 |
